## Installation

```
git clone git@github.com:donCelis/countries-api.git
cd countries-api
npm i
```

## Usage

### -Development server

```bash
npm run start
```

You can view the development server at `localhost:8080`.

### -Production build

```bash
npm run build
```

### -Deploy Github Pages

```bash
npm run deploy
```
You need edit publicPath variable in the file `webpack.config.js`
